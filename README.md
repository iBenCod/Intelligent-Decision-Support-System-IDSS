Project Description
***
An AI-driven Decision Support System (IDSS) designed to help businesses predict future sales trends and make data-driven strategic decisions.

Key Features

* Smart Forecasting: Uses Machine Learning models (like Prophet) to analyze historical data, seasonal patterns, and external factors for accurate predictions.
* Full Arabic Support: Provides a localized experience with an Arabic interface, reports, and visual analytics, making it accessible for users in the region.
* Interactive Dashboards: Features user-friendly visualizations and KPIs created with Streamlit to help managers interpret complex data easily.
* Data Management: Includes a complete pipeline for uploading, cleaning, and processing CSV/Excel sales records.
